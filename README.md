# Hamza Jawaid Portfolio

A free, one-page GitHub Pages portfolio microsite for Hamza Jawaid × Janobi Agency.

## Live URL after deployment
`https://hjbehari.github.io/hamza-portfolio/`

## Files
- `index.html` — one-page website structure
- `style.css` — premium dark editorial UI
- `script.js` — animations, case-study modal, email helper, mobile nav
- `assets/` — images and portfolio deck

## Deploy on GitHub Pages
1. Create a new repository named `hamza-portfolio` under the GitHub account `hjbehari`.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. Wait 1–3 minutes, then open:
   `https://hjbehari.github.io/hamza-portfolio/`
